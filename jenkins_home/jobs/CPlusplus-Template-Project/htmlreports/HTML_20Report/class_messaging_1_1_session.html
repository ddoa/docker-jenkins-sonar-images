◆ readMessage()

void Messaging::Session::readMessage ( )
inlineprotected

readMessage will read the message in 2 a-sync reads, 1 for the header and 1 for the body. After each read a callback will be called that should handle the stuff just read. After reading the full message handleMessageRead will be called whose responsibility it is to handle the message as a whole.
