◆ handleNotification()

virtual void Base::Observer::handleNotification ( )
pure virtual

A Notifier will call this function if this Observer will handle the notifications of that Notifier. It is the responsibility of the Observer to filter any events it is interested in.

Implemented in Model::Robot, View::RobotWorldCanvas, View::RobotShape, View::GoalShape, View::WayPointShape, View::RectangleShape, and View::LineShape.
