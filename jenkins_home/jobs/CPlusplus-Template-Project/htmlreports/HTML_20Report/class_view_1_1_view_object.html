Stops observing the current ModelObject, starts observing the new ModelObjects and displays it by calling ViewObject::resetViewObject

This function will call ModelObject::addReference. It will stop handling the notifications for the old object, start handling the notifications for the new object and reset this ViewObject (derived) class.
